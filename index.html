Language for Website Creation

Parlez-vous HTML?

When we think of language, we generally think of the words we use to communicate with each other on a daily basis, whether spoken or written. Perhaps we think of foreign languages, like French or Spanish. However, we also need language to communicate with machines. (After all, our computers aren't yet able to read lips.) This is where coding languages like HTML come in. HTML stands for Hypertext Markup Language and is a standardized system used to create web pages that can include a variety of effects from font and color to hyperlinks. An essential compontent to HTML are semantic elements. Semantic elements are various tags applied to pieces of text that provide meaning to both the developer and the browser. Simply put, semantic elements define the various parts of a webpage so that it can be understood by both man and machine. Let's take a look at some examples!

Basic HTML Semantic Elements

  1. All semantic elements require the use of < >, or angle brackets.
  2. Common elements include < header >, < body >, and < p > for paragraph.
  3. Many other tags allowing for font and color changes exist as well.

Fun Fact!

The word semantic is derived from Greek and means "signify".
